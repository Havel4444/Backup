Aquí está la ecuación que estamos resolviendo:
$$x + x + 1 + x + 2 = 36$$

Pasos de la solución:
\begin{align*}
x + x + 1 + x + 2 &= 36 \\
3x + 3 &= 36 \\
3x &= 36 - 3 \\
3x &= 33 \\
x &= \frac{33}{3} \\
x &= 11
\end{align*}

La solución es $x=11$.
